FAQ

What is TOML?

TOML (Tom's Obvious, Minimal Language) is a configuration file format that's easy to read and write. It's used by Cargo, PyPI, and many modern tools.

Is my data secure?

Yes! All TOML processing happens entirely in your browser using WebAssembly. Your configuration files never leave your device.
